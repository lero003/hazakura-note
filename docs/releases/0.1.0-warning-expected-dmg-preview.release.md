# hazakura-note 0.1.0 Warning-expected DMG Preview

This is a separate warning-expected DMG preview. It is not the `v0.1.0` source-only developer preview release.

This DMG is a developer-preview convenience artifact for macOS Apple Silicon / `aarch64`. It is ad-hoc signed only. It is not Developer ID signed, not notarized, not installer-grade, and macOS Gatekeeper warnings or rejection are expected.

Only use this preview if you are comfortable evaluating unsigned and not-notarized macOS preview binaries.

## Local Verification Before Publication

```bash
npm ci
npm run build:vite
cargo fmt --manifest-path src-tauri/Cargo.toml -- --check
cargo test --manifest-path src-tauri/Cargo.toml
npm run build:dmg-preview
npm audit
cargo audit
hdiutil verify src-tauri/target/release/bundle/dmg/hazakura-note_0.1.0_aarch64-warning-expected.dmg
shasum -c src-tauri/target/release/bundle/dmg/hazakura-note_0.1.0_aarch64-warning-expected.dmg.sha256
```

The DMG was mounted locally. The contained `hazakura-note.app` launched from the mounted volume, showed a window, exposed native File and View menus, and kept File > Save disabled in the no-file state.

`cargo audit` completed with warnings from transitive Tauri/wry dependencies, mostly GTK3/gtk-rs Linux stack unmaintained advisories plus `unic-*` and `glib` warnings. No high or critical vulnerability finding is treated as a blocker for this macOS warning-expected DMG preview.

## Boundary

Do not treat this as a signed, notarized, trusted, installer-grade, stable, or production-ready distribution.

The `v0.1.0` source-only release remains source-only. This preview exists as a distinct binary preview lane.
