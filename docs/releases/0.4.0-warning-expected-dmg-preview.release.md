# hazakura-note 0.4.0 Warning-expected DMG Preview

This is the v0.4 Markdown Review Navigation developer preview for testing hazakura-note on another Mac.

This DMG is a convenience artifact for macOS Apple Silicon / `aarch64`. It is ad-hoc signed only. It is not Developer ID signed, not notarized, not installer-grade, and macOS Gatekeeper warnings or rejection are expected.

Only use this preview if you are comfortable evaluating unsigned and not-notarized macOS preview binaries.

## What Changed Since 0.3.0

- Markdown files now have a current-file Outline pane with click-to-jump navigation.
- The status bar and scroll-position HUD show current Markdown section context without workspace-wide indexing.
- Markdown diff reviews show nearby heading context before changed blocks when ATX headings are available.
- Markdown preview links can open supported workspace text files when they resolve inside the selected workspace.
- Open tabs can be focused with Cmd+Option+Left / Cmd+Option+Right.
- Markdown preview readability was improved for lists, task checkboxes, horizontal rules, blockquotes, code blocks, and tables.
- Dark and Yakou theme contrast was tightened for Markdown editor highlighting and preview emphasis.
- Workspace file rows can reveal files in Finder.
- Diff compare-text eligibility feedback was improved for unsupported files.
- Diff, Outline, Preview, and Agent remain bounded review/navigation surfaces. This release does not add Git integration, LSP, project-wide indexing, arbitrary command execution, auto-apply, or auto-commit behavior.

## Expected macOS Behavior

Because this preview is not Developer ID signed or notarized, macOS may:

- warn that the developer cannot be verified
- block opening from a normal double-click
- require manual security override by an evaluator who understands the risk
- reject the app in Gatekeeper assessment

These warnings are expected. They are not evidence that this artifact is distribution-grade.

## Local Verification Before Publication

```bash

npm run build:vite
cargo fmt --manifest-path src-tauri/Cargo.toml -- --check
cargo test --manifest-path src-tauri/Cargo.toml
npm audit --audit-level=moderate
cargo audit --file src-tauri/Cargo.lock

SKIP_BUILD=1 npm run build:dmg-preview
```

The built local app reported version `0.4.0`, bundle identifier `lab.hazakura.note`, passed `codesign --verify --deep --strict --verbose=2`, launched locally from `src-tauri/target/release/bundle/macos/hazakura-note.app`, and was then quit.

The DMG was mounted locally. The contained `hazakura-note.app` reports version `0.4.0`, bundle identifier `lab.hazakura.note`, and passes `codesign --verify --deep --strict --verbose=2`.
