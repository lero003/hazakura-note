# hazakura-note 0.3.0 Warning-expected DMG Preview

This is the v0.3 Safe Editor developer preview for testing hazakura-note on another Mac.

This DMG is a convenience artifact for macOS Apple Silicon / `aarch64`. It is ad-hoc signed only. It is not Developer ID signed, not notarized, not installer-grade, and macOS Gatekeeper warnings or rejection are expected.

Only use this preview if you are comfortable evaluating unsigned and not-notarized macOS preview binaries.

## What Changed Since 0.2.0

- Diff is now a dedicated main workbench mode instead of only a right-side pane.
- Workspace text files can be compared explicitly by choosing separate source and target slots, then pressing Compare.
- The comparison surface uses a GitHub-PR-like split layout with source and target line numbers, added/removed markers, and file/workspace wording.
- Active editor changes, recoverable drafts, and external-change conflicts can be reviewed against disk before choosing what to keep.
- Diff and change review remain non-Git features. The app does not inspect Git status, branches, staging, commits, history, or repository state.
- Agent Workbench remains optional and explicit, while Safe Editor Mode stays the primary default boundary.

## Expected macOS Behavior

Because this preview is not Developer ID signed or notarized, macOS may:

- warn that the developer cannot be verified
- block opening from a normal double-click
- require manual security override by an evaluator who understands the risk
- reject the app in Gatekeeper assessment

These warnings are expected. They are not evidence that this artifact is distribution-grade.

## Local Verification Before Publication

```bash
npm ci
npm run build:vite
cargo fmt --manifest-path src-tauri/Cargo.toml -- --check
cargo test --manifest-path src-tauri/Cargo.toml
npm audit --audit-level=moderate
cargo audit --file src-tauri/Cargo.lock
npm run build
SKIP_BUILD=1 npm run build:dmg-preview
cd src-tauri/target/release/bundle/dmg
shasum -c hazakura-note_0.3.0_aarch64-warning-expected.dmg.sha256
```

The DMG was mounted locally. The contained `hazakura-note.app` reports version `0.3.0`, bundle identifier `lab.hazakura.note`, and passes `codesign --verify --deep --strict --verbose=2`.

`spctl -a -vv -t open` rejected the app with `source=Insufficient Context`, which is expected for this ad-hoc signed and not-notarized preview.

`npm audit --audit-level=moderate` passed with 0 vulnerabilities.

`cargo audit --file src-tauri/Cargo.lock` completed with the known Tauri/wry transitive warnings for GTK3/gtk-rs, `glib`, `proc-macro-error`, and `unic-*`. These remain dependency-refresh review items and are not treated as blocking this macOS warning-expected DMG preview.

Remote verification downloaded the published GitHub Release assets, confirmed `shasum -c` passed, `hdiutil verify` passed, and the mounted app reported version `0.3.0` / bundle identifier `lab.hazakura.note` with `codesign --verify --deep --strict --verbose=2` passing.
