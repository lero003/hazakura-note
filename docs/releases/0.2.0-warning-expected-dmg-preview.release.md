# hazakura-note 0.2.0 Warning-expected DMG Preview

This is the v0.2 Safe Editor developer preview for testing hazakura-note on another Mac.

This DMG is a convenience artifact for macOS Apple Silicon / `aarch64`. It is ad-hoc signed only. It is not Developer ID signed, not notarized, not installer-grade, and macOS Gatekeeper warnings or rejection are expected.

Only use this preview if you are comfortable evaluating unsigned and not-notarized macOS preview binaries.

## What Changed Since 0.2.0-pre.1

- Theme lineup was refined around System / Light / Dark / Sakura / Yakou / Shokou. The Hazakura Nexus variant was removed from the v0.2 release line.
- Shokou was brightened away from the earlier purple base toward a clearer blue/orange editorial tone.
- Japanese UI copy was polished for editor chrome, status/recovery banners, and Agent Workbench boundary surfaces.
- v0.3 planning was moved to Safe Editor Non-Git Diff / Review so the product promise "check with diff" is the next main milestone.
- Recurring automation guidance now follows a safe-editor review loop instead of making Agent Workbench the primary next lane.

## Expected macOS Behavior

Because this preview is not Developer ID signed or notarized, macOS may:

- warn that the developer cannot be verified
- block opening from a normal double-click
- require manual security override by an evaluator who understands the risk
- reject the app in Gatekeeper assessment

These warnings are expected. They are not evidence that this artifact is distribution-grade.

## Local Verification Before Publication

```bash
npm ci
npm run build:vite
cargo fmt --manifest-path src-tauri/Cargo.toml -- --check
cargo test --manifest-path src-tauri/Cargo.toml
npm audit --audit-level=high
cargo audit --file src-tauri/Cargo.lock
npm run build:dmg-preview
cd src-tauri/target/release/bundle/dmg
shasum -c hazakura-note_0.2.0_aarch64-warning-expected.dmg.sha256
```

The DMG was mounted locally. The contained `hazakura-note.app` reports version `0.2.0`, bundle identifier `lab.hazakura.note`, and passes `codesign --verify --deep --strict --verbose=2`.

The built app launched locally from the generated app bundle and was quit after the smoke with no `hazakura-note` process left running.

`spctl -a -vv -t open` rejected the app with `source=Insufficient Context`, which is expected for this ad-hoc signed and not-notarized preview.

`npm audit --audit-level=high` passed with 0 vulnerabilities.

`cargo audit --file src-tauri/Cargo.lock` completed with the known Tauri/wry transitive warnings for GTK3/gtk-rs, `glib`, `proc-macro-error`, and `unic-*`. No new high or critical macOS release blocker is treated as blocking this warning-expected DMG preview.
