# hazakura-note 0.5.0 Warning-expected DMG Preview

> **Pi CLI Provider, Image Assets, and Authoring Stability**

v0.5.0 adds `pi` as an allowlisted Agent Workbench CLI provider, improves terminal stability, and introduces image asset workflows for Markdown authoring.

This is a warning-expected macOS DMG preview. The app is ad-hoc signed and not notarized, so Gatekeeper warnings are expected.

Only use this preview if you are comfortable evaluating unsigned and not-notarized macOS preview binaries.

## What Changed Since 0.4.0

### Agent Workbench

- Added `pi` as an allowlisted Agent Workbench CLI provider alongside existing `codex` and `opencode`.
- Terminal resize is now debounced (100ms) to reduce backend calls.
- Improved session-stopped visual styling in the Agent pane.
- Agent session polling runs at 100ms for responsive TUI output.

### Image Assets

- Clipboard images (Cmd+V) are saved to `assets/<content-hash>.<ext>` and inserted as Markdown image references.
- Dragging image files from Finder into the window imports them into `assets/`.
- Hash-based deduplication: pasting the same image multiple times creates only one file.
- Pasted images render in the Markdown preview pane.
- HTML export inlines assets images as data URIs.
- When no workspace folder is open, `assets/` is created next to the active file.

### Export

- HTML export generates a standalone HTML file via the save dialog.
- Print to PDF opens the document in the system browser for printing (fallback from `window.print()`).

### Authoring Polish

- **Zen Mode** (Cmd+Shift+F): hides all chrome, centers the editor. Escape exits.
- **Spellcheck Toggle** (Cmd+Option+;): enables or disables native macOS spellcheck in the editor.
- **Table Insertion** (Cmd+Shift+T or toolbar button): inserts a 3-column Markdown table.

### Stability & UX

- Export menu items are always enabled (fixed macOS native menu sync issue).
- Image paste and drag-drop work without a pre-selected workspace folder.
- Image status messages are localized for Japanese UI.
- Zen mode centers the editor correctly with collapsed file-tree column.

## Known Limitations

- Print to PDF uses a browser print fallback (macOS print dialog may not appear in all Tauri v2 / WKWebView configurations).
- Image paste currently inserts Markdown syntax without a dedicated alt text prompt.
- Image deduplication uses a content hash (FNV-1a) for local convenience; it is not a cryptographic integrity feature.
- The JavaScript bundle remains large (~1.3 MB / 417 KB gzip); code splitting is planned for a future release.
- This release does not add Git integration, LSP, project-wide indexing, arbitrary command execution, auto-apply, or auto-commit behavior.

## Expected macOS Behavior

Because this preview is not Developer ID signed or notarized, macOS may:

- warn that the developer cannot be verified
- block opening from a normal double-click
- require manual security override by an evaluator who understands the risk
- reject the app in Gatekeeper assessment

These warnings are expected. They are not evidence that this artifact is distribution-grade.
